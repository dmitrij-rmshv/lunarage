О сайте!

Сайт-калькулятор для определения лунного возраста и предстоящих «лунных юбилеев»

Вычисления носят упрощённый характер, за опору приняты данные по времени лунных циклов, сгенерированные с помощью ресурса Astronet.

Диапазон допустимых для ввода дат: с февраля 1931 года по настоящее время. В каптче можно вводить маленькие английские буквы.

Сайт разработан, запущен и поддерживается Дмитрием Ромашевым.
В ближайших планах разработка функционала обратной связи для сбора пожеланий, рекомендаций и откликов.
